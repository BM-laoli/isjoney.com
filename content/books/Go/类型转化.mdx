# 章节概述如下


> 本章节 涉及的语法 不多，golang中不支持 隐式类型的转化 只能手动
>



```go
package main
import "fmt"

// 所谓的类型转化非常的容易理解type_name(expression)就完了，
func main()  {
	// 接下来的测试是使 int ===> float 类型.如果有精度丢失的问题，可以另外考虑
	var sum int =17
	var count int = 6
	var mean float32
	
	mean = float32(sum) / float32(count)
	fmt.Printf("mean 的值为: %f\n",mean)
}
```
