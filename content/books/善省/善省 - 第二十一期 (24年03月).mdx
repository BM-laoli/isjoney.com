# 关于高效学习的话题
我的观点：  
👍 对每个问题都应该如此这般分析  
1. 是什么东西  
2. 使用了什么技术或者方案  
3. 解决什么问题



比如：关于RXJS的操作符

| 是什么东西 | 使用了什么样的技术或方案 | 解决了什么样的问题 |
| --- | --- | --- |
| 他们都是创建source相关的操作符 | **<font style="color:rgb(51, 51, 51);background-color:rgb(248, 248, 248);">Rx.Observable.create</font>****<font style="color:rgb(37, 41, 51);"></font>** | 从无到有创建数据 |
| | **<font style="color:rgb(51, 51, 51);background-color:rgb(248, 248, 248);">Observable.of(fish1,fish2);</font>** | 数据是分散的，用这个of可以把它们组转在一起 |
| | **<font style="color:rgb(51, 51, 51);background-color:rgb(248, 248, 248);">var fishes = [fish1, fish2];</font>**<br/>**<font style="color:rgb(51, 51, 51);background-color:rgb(248, 248, 248);">var </font>****<font style="color:rgb(0, 134, 179);background-color:rgb(248, 248, 248);">source</font>****<font style="color:rgb(51, 51, 51);background-color:rgb(248, 248, 248);"> = Observable.from(fishes);</font>** | 用特定数据结构Array把他们组装在一起 |
| | **<font style="color:rgb(51, 51, 51);background-color:rgb(248, 248, 248);">x.Observable.fromEvent(document.body, </font>****<font style="color:rgb(221, 17, 68);">'click'</font>****<font style="color:rgb(51, 51, 51);background-color:rgb(248, 248, 248);">);</font>** | 用来从事件监听出，触发数据的创建 |
| | **<font style="color:rgb(51, 51, 51);background-color:rgb(248, 248, 248);">empty,nerver,throw</font>** | 特定场景下的数据创建方式 |
| | **<font style="color:rgb(51, 51, 51);background-color:rgb(248, 248, 248);">Rx.Observable.interval(1000) </font>** | 间隔时间的创建数据 |




依照此法 ，我们可以运用到更多的领域中去！这将会让你的学习效率可观的上升
