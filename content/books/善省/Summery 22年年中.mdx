原因嘛，基本上大差不差，总结一下就这样几句话  
**<u><font style="color:#FA541C;">“毅力太差，容易受到干扰，没有能坚持完成自己既定的计划，比如学英语从3月份开始，几乎没有认真的再学习了。”</font></u>**

# 改进措施<font style="color:#E8323C;"></font>


先看工作上的：

- [x] 首先是这个时间上，建议直接和对接的业务方沟通，在需求的分析和技术预研上多要一些时间，把它提升到30%，避免做 “改中做 做中改，频繁改的项目”
- [x] 管理上的要求：每提出一个方案，都要有完整的Plan，这样拿到这份Plan的老板，才能做出准备的判断，是否是值得的，它的价值能否被体现，所以一点要有Plan ，**<font style="color:#FA541C;">“人人都是产品经理”</font>**
- [x] 人际上：很多时候，我们不能妥协，我这个人就是太容易 “软了不了商量商量来，这种模式，至少在现在的团队上行不通的（有教训），**<font style="color:#FA541C;">软硬兼施，篇硬为主</font>**，我觉得对于我现在待的团队，这种方式更加容易出结果”



再看业余学习上的：

- [x] **<font style="color:#FA541C;">缩小</font>**自己计划上的东西，不要在同一时间内 去做很多不相关的事情。长期且专注且深耕，对于一名技术人来说，太重要了
- [x] 做事情一定要有结果！



最后看个人生活上的：

# 下半年目标
> 总体来说，下半年就一句话：**<font style="color:#FA541C;">“缩小目标，缩小范围，所有的所有都求稳，求往扎实，往深处札根，不管是技术还是其他”</font>**
>

如果说自己除去学习英语什么都不干这显然是不可能的，比如在现在的高压情况下，英语EF 确实需要迫切的学习，但是_**<u>面试八股 和 算法题必须要看</u>**_，这个也是一个长期的事情。

先看工作上的：

- [x] 只有一个想法💡 ，Shopify项目做完，把监控平台计划给出来，然后把其他 构建系统什么的，都给罗列出来，一点一点地去构建这个体系。很难但是只要做了，就不怕难，就怕你不做却说难。
- [x] 求稳 ，但是也要保持警惕，裁员才是刚刚开始。一周至少需要，花费1-2h在 八股文 这件事情上；修改简历这件事情，下周结束前要准备好
