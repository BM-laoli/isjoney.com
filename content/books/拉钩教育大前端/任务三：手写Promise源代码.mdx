## 一、概述
 	 要手写promise 就得需要了解，Promise 的基础用法，以及它上面的一些细节，比如 Promise是一个构造函数（或者说是一个类因为可以new Promise）,它返回一个对象，对象上有then方法 有cath方法。Promise也有静态方法，reject ,resolve 等

Promise是状态的，所以我们需要 了解它的几个状态，才好实现它，比如 fail  success pending什么的

前期呢，我妈妈需要了解的东西和需要准备的东西是：

1. Promise 就是一个类 在执行这个类的时候 需要传递一个执行器进去 执行器会立即执行
2. Promise 中有三种状态 分别为 成功 fulfilled 失败 rejected 等待 pending  
pending -> fulfilled  
pending -> rejected  
一旦状态确定就不可更改
3. resolve和reject函数是用来更改状态的  
resolve: fulfilled  
reject: rejected
4. then方法内部做的事情就判断状态 如果状态是成功 调用成功的回调函数 如果状态是失败 调用失败回调函数 then方法是被定义在原型对象中的
5. then成功回调有一个参数 表示成功之后的值 then失败回调有一个参数 表示失败后的原因

## 二、具体实现
### 1.首先是实现基础逻辑(核心)
```javascript

class MyPromise {
  constructor (executor) {
      executor(this.resolve, this.reject)
  }
  // promsie 状态 
  status = PENDING;
  // 成功之后的值
  value = undefined;
  // 失败后的原因
  reason = undefined;

  resolve = value => {
    // 如果状态不是等待 阻止程序向下执行
    if (this.status !== PENDING) return;
    // 将状态更改为成功
    this.status = FULFILLED;
    // 保存成功之后的值
    this.value = value;
  }
  reject = reason => {
    // 如果状态不是等待 阻止程序向下执行
    if (this.status !== PENDING) return;
    // 将状态更改为失败
    this.status = REJECTED;
    // 保存失败后的原因
    this.reason = reason;
  }
  then (successCallback, failCallback) {
    // 状态判断
    if( this.status === FULFILLED ){
      successCallback(this.value)
    }else if( this.status === REJECTED ){
      failCallback(this.reason)
    }
  }
}
```

### 4. 处理链式调用
then方法应该返回promise对象，如果不返回默认返回一个prmomis如果是普通值就直接调用promise 的resolve反方包裹之后的形成promise对象的一个回调，如果是返回promise就得看这个 promsie的状态，如果这个状态是成功的 就resolve 否则就调reject
